# OpenDiabetes Atlas

OpenDiabetes Atlas is a beginner-friendly, open-source project focused on diabetes awareness and advocacy.

This project was created to make diabetes easier to understand for non-medical audiences, and to highlight how it affects not just individuals, but entire families.

## What this project includes

- Plain-language explanations of diabetes
- Common myths vs facts
- Family and caregiver perspectives
- Open, reusable educational content

## Who this is for

- Students learning about diabetes
- Families and caregivers
- Educators
- Anyone who wants clear, respectful information

## Contributing

This project is open to contributions, especially:
- Clarifying explanations
- Fixing typos
- Adding sources
- Improving accessibility

No medical advice is provided.
